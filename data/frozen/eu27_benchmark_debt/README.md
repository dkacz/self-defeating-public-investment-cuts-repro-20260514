# EU27 Benchmark Debt Endpoint Input

This folder freezes the already accepted EU27 benchmark debt endpoint values used as the comparison row in the Mozdzen full replacement branch debt table.

Source artifact:

`artifacts/action_tasks/empirical_chapter_clean_repro_20260426/02_average_eu_debt/`

Source files:

- `results/dsa_summary_selected_years.csv`
- `results/direct_dy_summary.csv`

The copied values are the 2036 margins for the three annual public-investment actions of 1 percentage point of GDP in 2028, 2029 and 2030. They are included here only to synchronize the generated branch table and branch notebook with manuscript Table 5 and Appendix C; they do not replace the accepted clean-repro source of truth.
