# Frozen Adopted Run Outputs

This folder contains the adopted branch's frozen first-stage, Polish output/spending and debt-accounting outputs for `tiva2022_gfcf_realppp_networth`.

These files are copied into the branch so manuscript tables, figures and the branch notebook are reproducible from local frozen artifacts instead of upstream paths outside this directory. The source-level state variables are rebuilt separately from `data/frozen/adopted_sources/`; these run outputs document the adopted estimation and downstream accounting results used by the current branch draft.

Hashes are written to `data/provenance/adopted_branch_frozen_inputs_manifest_20260514.csv` by `code/run_full_replacement_repro.py`.
